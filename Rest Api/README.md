## Installation

```bash
npm install
```

## Usage

### Start the Server
```bash
npm start
```

Server will start on http://localhost:3000

### Run Tests
```bash
npm test
```

## API Endpoints

- **Airports:** `GET /api/airports`
- **Airlines:** `GET /api/airlines` (with filters: country, iata, icao, search)
- **Countries:** `GET /api/countries` (with filters: region, subregion, currency, language, search)
- **Country by Code:** `GET /api/countries/:code`

## Web Interface

- **Airports:** http://localhost:3000/
- **Airlines:** http://localhost:3000/airlines.html
- **Countries:** http://localhost:3000/countries.html
